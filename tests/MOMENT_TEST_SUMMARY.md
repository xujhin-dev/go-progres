## API 端点测试

### 认证相关
- `POST /auth/otp` - 发送验证码 ✅
- `POST /auth/login` - 用户登录 ✅

### 动态相关
- `POST /moments/publish` - 发布动态 ✅
- `GET /moments/feed` - 获取动态列表 ✅
- `PUT /moments/:id/audit` - 审核动态 ✅
- `POST /moments/:id/comments` - 添加评论 ⚠️
- `GET /moments/:id/comments` - 获取评论列表 ✅
- `POST /moments/like` - 点赞/取消点赞 ✅
- `GET /moments/topics` - 获取话题列表 ✅

## 技术实现

### 后端架构
- **框架**: Gin Web Framework
- **数据库**: PostgreSQL + Redis
- **认证**: JWT Token
- **架构**: 模块化设计，使用Repository模式

### 数据模型
- **Post**: 动态模型，支持多媒体内容和话题关联
- **Topic**: 话题模型
- **Comment**: 评论模型，支持层级结构
- **Like**: 点赞模型

### 存储实现
- 当前使用内存存储进行测试（SimpleMomentRepository）
- 生产环境需要实现真正的数据库持久化
